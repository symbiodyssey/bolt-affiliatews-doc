Related content¶

This function is only available for the architecture mode number three (Items, categories and reviews). For more information, check : Website architecture.

The extension “Related Content By Tags” (xiaohutai/bolt-relatedcontentbytags) allows you to add related content. Install it : Required extensions.

Set the desired tags for every item. For more information, check : Edit an item.

Finally, activate the mode in the theme.yml file.

In the dashboard, in the menu on the left go to File management -> view/edit templates -> affiliatews -> theme.yml

Set the variable related_content to true :

related_content: true
